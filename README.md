# projeto-GuliX
GuliX - E-commerce de Hardwares

API backend de um sistema de e-commerce.

## Status
-  Backend: Completo
-  Frontend: em desenvolvimento

## Principais tecnologias utilizadas:

## Back-End:
- Java + Spring Boot
## Banco de dados:
- MySQL
# Autenticação e Autorização:
- JWT/Spring Security

## Funcionalidades
- Registro e login de usuário
- Autenticação e autorização
- CRUD
- Sistema de pedidos
- Sistema de estoque
- Sistema de carrinho


## Como testar
- Postman / Insomnia
- (Opcional) Swagger
